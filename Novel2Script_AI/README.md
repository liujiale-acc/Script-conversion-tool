AI 影视剧本全链生产系统 (Script-Expert 2.0)

1. 项目愿景
本项目专为影视编剧与内容创作者打造，旨在通过 DeepSeek-V3 大模型的超强语义理解能力，将小说文本一键转化为包含**“角色画像、专业剧本、导演分镜提示词”在内的全链生产稿件，极大缩短影视前期创作周期。

2. 核心功能亮点 (2.0 更新)
- 【核心】智能剧本转换：自动识别对白、动作与场景，生成符合工业标准的剧本排版，彻底去除 Markdown 符号干扰。
- 【新增】角色画像分析：深度提取人物外貌、性格及核心驱动力，为选角和演员围读提供参考。
- 【新增】视觉分镜提示词：为每个场景自动生成可用于 Midjourney/SD 的 AI 绘画提示词，辅助导演构建画面感。
- 【优化】UI 沉浸式设计：双栏实时预览界面，支持创作类别（电影/短剧/动漫等）的一键切换。
- 【增强】无损 Word 导出：完美保留角色分析与分镜建议，生成直接可用的专业文档。

3. 技术架构
- 大模型引擎：DeepSeek-V3 (基于 OpenAI SDK 兼容接口)。
- 后端支撑：Python 3.9 + Flask 框架。
- 文档自动化：`python-docx` 实现高保真排版逻辑。
- 前端技术：HTML5 + CSS3 (Flexbox/Grid 响应式布局) + 异步 Fetch API。

4. 演进历程 (PR 提交轨迹证明)
*本项目严格按照议题规范进行分步开发与提交：*
- Phase 1 (v1.0): 完成 Flask 基础架构搭建，实现小说到剧本的基础格式转换逻辑。
- Phase 2 (v2.0): 
    - 引入结构化 Prompt 指令，实现角色与分镜提示词的并行生成。
    - 升级 UI 交互系统，增加状态反馈与专业审美优化。
    - 优化数据清洗引擎，解决 AI 输出中 Markdown 杂质残留问题。

5. 快速上手
环境配置
1. 克隆仓库：`git clone [你的仓库地址]`
2. 安装依赖：`pip install -r requirements.txt`
3. 配置环境变量：在 `.env` 文件中填入 `DEEPSEEK_API_KEY=您的密钥`

6. 启动应用
python app.py
